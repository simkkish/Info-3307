  1.  1.1 Generate Purchase Agreement and Compare Payment
    Case: Generate Salesperson Report
    2. Get the payment method from the customer (1.1) as input.
    If( payment method == Cash Collect payment), return the difference of cost price and payment to customer and generate the receipt.
    Else, Collect payment and generate & return the receipt to customer

  2.  1.1.1 Customer verification
    Case: Customer Verification
    3. Get details of customers and check the information
    If they are already in the system, Update that the information is verified
    Else, Create the new customer.

  3.  1.1.4 Process Payment
    Case: Process Payment
    4. Check if they made a payment or not
    If you receive the payment from customer. send the payment record to inventory file and provide the receipt to customer
    Else, Ask for payment again

  4.  1.2 Customer Returns Defective/Unwanted Equipment
    Case: Customer Returns Defective/ Unwanted Equipment
    5. Get the reason for returning the item.
    If the Customer is returning product because it is defective, Update the inventory file and send request for maintenance
    Else, Update the inventory file and stock the product

  5.  2.1 Process Returned Inventory
    Case: Process Returned Inventory
    6. Get the list of returned items from 1.2 as input.
    If the item is defective, send it to Suppliers.
    Else, Send it to inventory file.

  6.  2.2 Check for Ordering new Inventory
    Case: Check for Ordering new Inventory
    7. Get the current inventory records as of input from the inventory file.
    If (an item quantity &get; Minimum quantity to be present in stock), Order quantity needed to make the item quantity at optimum level
    Else, Do not order the item.

  7.  3.1 Generate Salesperson Report
    Case: Generate Salesperson Report
    8. Retrieve the monthly sales report from Sales report file, Separate the salesperson information from the sold items report
    Generate the monthly sales report for each salesperson, Update the salesperson information in employee file
    Give salesperson the monthly report with the information of total items sold in the month.

  8.  3.4 Generate Current Inventory Report
    Case: Generate current inventory report
    9. Get the record details from inventory file.
    If the record is valid, send it to the record file.
    Else, Ask for the valid record.
