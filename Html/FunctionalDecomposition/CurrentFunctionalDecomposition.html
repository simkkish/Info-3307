Consolidation of Business Functions:

  1. Provide Customer Service/Handle Transactions
    1. Manage Purchase
      1. Customer Verification
      2. Generate Purchase Agreement
      3. Equipment Sale
      4. Process Payment
    2. Manage Returns
      1. Verify Purchase
      2. Check for Defectiveness
      3. Verify return time
      4. Receive Equipment
      5. Process Refund
    3. Manage Maintainence
      1. Verify Customer Status
      2. Add Customer Information
      3. Update Customer Information
      4. Record Problem
      5. Estimate Time and Price
      6. Finalize Repair Service to Apply
      7. Record the Repair Service Applied
      8. Generate Service Entry Form
      9. Finalize Service Sale
  2. Maintain Inventory
    1. Order new inventory (equipment)
      1. Run Inventory report (monthly)
      2. Compare reorder request
      3. Adjust order quantity
      4. Calculate total order price
      5. Manage orders according to the suppliers
      6. Finalize and send orders
    2. Process new inventory
    3. Process returned inventory
  3. Generate Reports
    1. Generate sales report
    2. Generate salesperson sales/commission report
    3. Generate technician service report
    4. Generate current inventory report
    5. Generate service summary report
